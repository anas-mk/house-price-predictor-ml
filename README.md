# House Price Predictor (Machine Learning App)

A machine learning web application that predicts house prices based on user-provided features such as bedrooms, bathrooms, living area, house condition, and nearby schools.  
The app is built using **Python**, **Scikit-learn**, and **Streamlit**.

---

## Features
- Predict house prices instantly
- Clean and modern Streamlit UI
- Uses a trained machine learning regression model
- Simple and beginner-friendly project structure

---

## Machine Learning
- Model loaded using `joblib`
- Trained on housing data using regression algorithms (e.g. Random Forest / Decision Tree)
- Input features:
  - Bedrooms
  - Bathrooms
  - Living Area (sqft)

  - Nearby Schools
